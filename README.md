# PetProov Social Assets

Public static media assets for approved PetProov social posts.

This repository is used by the local Codex social workflow to host approved images and videos for Buffer scheduling.

## Rules

- Only approved assets should be committed here.
- Do not commit draft artwork, rejected artwork, credentials or private campaign notes.
- Keep paths stable after posting because Buffer may continue to reference these URLs.

## Suggested Asset Paths

```text
assets/
  instagram/
    2026/06/
      social-001.png
  linkedin/
    2026/06/
      social-001.png
```

Public GitHub Pages URLs should look like:

```text
https://katemargolis86-beep.github.io/petproov-social-assets/assets/instagram/2026/06/social-001.png
```
